Who is a Site Reliability Engineer?
A software engineer who uses code to manage the reliability of software systems.

Characteristics of a Good SRE?
SRE Metrics: Can tell whether their app is healthy or unhealthy in minutes.
Believes in Extreme Automation - Says no to doing the same thing manually more than twice.
Can define and implement Availability; aka SLIs, SLOs, Error Budgets.
Identify and build Reliability Fixes backlog for Dev teams.
Looks for failure scenarios and thinks failovers.
Thinks Continuous Improvement / Optimization.
Loves Logs.
